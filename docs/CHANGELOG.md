# Changelog

All notable changes to the AnonLink project will be documented in this file. The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/), and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2026-07-08

### Added
* **Onboarding Workflow**: Compulsory Age & Gender selection screens mapping to a privacy-first onboarding model.
* **Subscriptions & Feature Gating**: Reusable `IPaymentProvider` billing factory, REST route feature guards, and Socket.IO voice/video queue blocks.
* **Ephemeral Media Sharing**: Upload support for temporary images, auto-deletion schedulers (60s TTL), right-click blocker controls, and screenshot-deterrent CSS/JS on previews.
* **Admin Dashboard Features**: Manual subscription grants, global capability switches, and subscription transaction overviews.
* **Integration Tests**: Four automated test files checking onboarding options, user types, media TTLs, and plan validity logic.
